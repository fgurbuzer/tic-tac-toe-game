# Tic-Tac-Toe Game

A responsive Tic-Tac-Toe game built with Vue 2 that demonstrates component-based UI development, reactive state management, and basic game logic implementation.

## Overview

This project is a simple two-player Tic-Tac-Toe application where players take turns placing **X** and **O** on a 3x3 grid. The application detects winning combinations, handles draw conditions, and allows the board to be reset for a new game.

## Features

- Two-player turn-based gameplay
- Real-time winner detection
- Draw state detection
- Reset functionality
- Dynamic status updates
- Clean and minimal interface

## Tech Stack

- Vue.js 2
- JavaScript (ES6)
- HTML5
- CSS3

## Purpose

This project was developed to strengthen practical understanding of core Vue 2 concepts, including:

- component structure
- reactive data binding
- conditional rendering
- list rendering with `v-for`
- event handling
- computed properties
- game state management
